Документация pyyaru
===================

pyyaru — это Python-интерфейс для API блог-сервиса ya.ru, который можно использвать для написания собственных Я.ру-клиентов.

* pyayru предназначен для разработчиков.
* pyyaru переплюнул API Я.ру и находится в стадии глубокой альфы.
* pyyaru не умеет делать того, чего не умеет делать API Я.ру и даже больше.

Репозитарий проекта — http://github.com/idlesign/pyyaru.
